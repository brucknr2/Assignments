Abstract    :
                                                                                             
              During Phase 1 of the system programming for the conversion                    
              of the NSF Reviewer system from the HP to the IBM 3090 the                     
              contractor will perform the following tasks:                                   
                                                                                             
                   o  Develop program communications architecture, load                      
                   test data for program testing purposes, interface with                    
                   Menu Processor, generate Reviewer System error                            
                   messages, program and unit test all program according                     
                   to detailed design specifications.                                        
                                                                                             
                   o  Perform integration and full range of system                           
                   testing using system test plans.  Modify software as                      
                   needed.                                                                   
                                                                                             
                   o  Program, unit test, and system test the conversion                     
                   programs.                                                                 
                                                                                             
                   o  Develop use and operations manuals and training                        
                   program for users.                                                        
                                                                                             
                   o  Install software in parallel with HP.  Verify and                      
                   correct software based on parallel results.                               
                                                                                             
                                                                                             
              It is recommended that the first incremental funding of                        
              $62,075 be awarded.
