Abstract    :
               Work is performed to improve the dependability of large-scale  concurrent
              software by; (1) devising scalable, compositional analysis  techniques, (2)
              developing a sound framework for combining  verification, analysis and testing
              techniques, and (3) integrating  analysis techniques in software development
              environments.
